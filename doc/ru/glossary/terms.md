# Глоссарий

Термины, используемые в экосистеме CellFrame.

*   **Cell (Ячейка)**: Логический раздел сети/блокчейна.
*   **Chain (Цепочка)**: Последовательность атомов (блоков или событий).
*   **Atom (Атом)**: Минимальная единица данных в цепочке. Может быть блоком или событием.
*   **Datum (Датум)**: Единица полезной нагрузки (например, транзакция, декларация токена) внутри Атома.
*   **Ledger (Леджер)**: База данных всех неистраченных выходов транзакций (UTXO) и балансов.
*   **Mempool (Мемпул)**: Временное хранилище для транзакций, ожидающих включения в цепочку.
*   **GDB (Global Database)**: Распределенное хранилище ключ-значение, синхронизируемое по сети.
*   **CondTx (Условная транзакция)**: Транзакция, требующая выполнения определенных условий для траты выхода (например, временная блокировка, предоставление услуги).
