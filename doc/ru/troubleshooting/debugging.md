# Отладка

Как отлаживать Python приложения.

## Логирование

Используйте `dap.log_it` или `CellFrame.logIt` (устаревший алиас) для записи в лог приложения.

```python
from CellFrame import logIt

logIt.debug("Отладочное сообщение")
logIt.info("Информационное сообщение")
logIt.error("Сообщение об ошибке")
```

Расположение логов зависит от настроек инициализации.

## Интерактивная отладка (pdb)

Вы можете использовать стандартные инструменты отладки Python.

1.  Вставьте `import pdb; pdb.set_trace()` в ваш python код.
2.  Запустите скрипт: `python3 my_script.py`.
3.  Выполнение остановится на точке останова.

## Segmentation Faults (Сбои сегментации)

Если приложение падает (Segfault):
1.  Включите дампы памяти: `ulimit -c unlimited`.
2.  Запустите приложение до падения.
3.  Проанализируйте дамп с помощью `gdb`:
    ```bash
    gdb python3 core
    bt
    ```
4.  Ищите кадры python в обратной трассировке (stack trace).
