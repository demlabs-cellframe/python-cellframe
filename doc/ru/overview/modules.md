# Основные модули

SDK разделен на несколько основных модулей.

## Модуль CellFrame

Это основная точка входа для блокчейн-логики.

*   **CellFrame.init() / deinit()**: Инициализация и очистка.
*   **CellFrame.set_log_level()**: Настройка логирования.

### CellFrame.chain
Функционал, связанный со структурой блокчейна.
*   `Wallet`: Управление ключами и адресами.
*   `TX`: Создание и парсинг транзакций.
*   `Ledger`: Доступ к леджеру (история транзакций, балансы).
*   `Chain`: Управление блоками и атомами.

### CellFrame.network
Сеть и коммуникация узлов.
*   `Net`: Конфигурация сети.
*   `Node`: Взаимодействие с удаленными узлами.

### CellFrame.services
Сервисы, работающие на ноде.
*   `Service`: Базовый класс для создания пользовательских сервисов.

### CellFrame.common
Общие утилиты и типы.
*   `Datum`: Базовая единица данных в цепочке.

## Модуль DAP (`python-dap`)

Низкоуровневый системный доступ.

### dap.core
*   `config`: Парсинг конфигурационных файлов.
*   `log_it`: Функция логирования.

### dap.crypto
*   `HashFast`: Хэш-функции.
*   `Sign`: Цифровые подписи.
*   `CryptoKey`: Управление ключами.

### dap.global_db
*   Доступ к Global Database (GDB) — распределенному KeyValue хранилищу.
