# Glossary

Terms used in CellFrame ecosystem.

*   **Cell**: A logical partition of the network/blockchain.
*   **Chain**: A sequence of atoms (blocks or events).
*   **Atom**: The minimal unit of data in a chain. Can be a block or an event.
*   **Datum**: A unit of data payload (e.g., a transaction, a token declaration) inside an Atom.
*   **Ledger**: The database of all unspent transaction outputs (UTXO) and balances.
*   **Mempool**: A temporary storage for transactions waiting to be included in a chain.
*   **GDB (Global Database)**: A distributed key-value storage synchronized across the network.
*   **CondTx (Conditional Transaction)**: A transaction that requires specific conditions to be met to spend its output (e.g., time lock, service delivery).
